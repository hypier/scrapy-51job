# 招聘爬虫

## 功能

- 此爬虫采用scrapy框架，以redis作为地址缓存，以ES作为内容存储
- 可以单独启用也可以导入[crawlab](https://github.com/crawlab-team/crawlab) 平台

## 爬虫

- 51job
- 猎聘
- 拉勾
- 智联
- Boss直聘：_易被反爬_
